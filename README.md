Описание проекта

Этот проект представляет собой локальный поисковый движок. 
Приложение осуществляет поиск по текстовым документам, используя инвертированный индекс для быстрого нахождения релевантных документов.
Принципы работы поискового движка :
1.  В конфигурационном файле перед запуском приложения задаются названия
файлов, по которым движок будет осуществлять поиск.
2. Поисковый движок должен самостоятельно обходить все файлы и
индексировать их так, чтобы потом по любому поисковому запросу находить наиболее
релевантные документы.
3. Пользователь задаёт запрос через JSON-файл requests.json. Запрос — это
набор слов, по которым нужно найти документы.
4. Запрос трансформируется в список слов.
5. В индексе ищутся те документы, на которых встречаются все эти слова.
6. Результаты поиска ранжируются, сортируются и отдаются пользователю,
максимальное количество возможных документов в ответе задаётся в
конфигурационном файле.
7. В конце программа формирует файл answers.json, в который записывает
результаты поиска.

Установка 
git clone https://github.com/Andrey34551/skillbox_project.git

cd search_engine
Сборка проекта
# Создайте папку для сборки
mkdir build
cd build

# Генерация проектных файлов
cmake ..

# Сборка проекта
cmake --build .
